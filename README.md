# Instalation instruction
- Firstly install `lua-lgi` module. On linux, just install it directly from package manager (e.g: `sudo dnf install lua-lgi` on fedora). On window, you should install follow the instruction from the [origin of this plugin](https://github.com/jereyes4/xournalpp-bookmarks) Thanks for the wonderful plugin jereyes4!
- Next, grab the bookmark plugin in the release -> extract it and then throw the `AA_Bookmarks` directory into xournalpp plugin directory ([location](https://xournalpp.github.io/guide/file-locations/)). Then open xournalpp -> plugin (top panel), enable `AA_Bookmarks` plugin -> close and reopen xournalpp. Enjoy!
